# Fiesta Heroes Documentation

Welcome to the Fiesta Heroes development documentation repository! This repository contains the documentation for developing Fiesta Online.
A live version of the documentation can be found at [https://doc.fiestaheroes.com](https://doc.fiestaheroes.com).

# Getting Started

To start local development and contribute to the documentation, follow these steps:

1. Clone the repository to your local machine:
```bash
git clone https://github.com/FiestaHeroes/Documentation.git 
```
2. Navigate to the repository and install the dependencies:
```bash
cd fiesta-heroes
npm install
```
3. Start the local development server:

```bash
npm run start
```
4. The documentation site should now be running locally at [http://localhost:3000](http://localhost:3000).
   
# Contributing

If you would like to contribute to the documentation, please open a pull request with your changes. If you are unsure about something, feel free to open an issue to discuss it or join our [discord server](https://discord.com/invite/TCFzBPgtPX).

### Adding a new page

To add a new page to the documentation, follow these steps:

1. Create a new markdown file in the `docs` directory. The file name will be the URL path of the page. For example, `docs/new-page.md` will be available at `/new-page`.
2. Add the following frontmatter to the top of the file:

```md
---
title: New Page 
sidebar_position: 0  # Position of the page in the sidebar.
---

# CONTENT HERE
```
3. Add the content for the page below the frontmatter.
4. Commit your changes and open a pull request.

### Lua 

For any changes to the lua documentation, please modify the fiesta.lua file located in the static/files directory. The documentation under `docs/Lua Scripting` is auto-generated by a script located at scripts/convert_lua.py.

To generate the markdown from the Lua file, follow these steps:

1. Have python3 installed
2. Run the following commands:

```bash
cd scripts
python convert_lua.py
```

The script will generate the markdown and place it in the correct directory.
